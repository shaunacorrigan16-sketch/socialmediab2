# Social Media : B2


#### In this B2 level English course, we discuss the topic of social media. We will discuss:
- How to use social media responsibly and protect your privacy
- What fake news is and how to detect it
- Role models on social media
- "Cancel culture" and
- Social media and self-esteem

#### Through this, we will learn:
- Key vocabulary related to social media platforms
- What **modal verbs** are, and how to use them to give advice
- What **phrasal verbs** are, and how to use some examples.
- How to write an **opinion piece** text.
